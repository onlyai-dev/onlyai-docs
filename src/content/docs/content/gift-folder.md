## **Gift Folder**

Gift Folder is used to send free content to fans in specific situations.

![](/images/docs/gift-folder/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-06-04%20%D0%BE%2016.21.23.png)

To create a Gift Folder, go to **Content → Gifts** and click **Add Folder**.

When creating a folder, you need to:

- select media files via **Select Media**
- enter a folder name
- fill in the description and other parameters

For faster setup, you can use the **AI Fill** feature, which automatically generates a description and fills in the required fields based on the selected content.

![](/images/docs/gift-folder/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-06-04%20%D0%BE%2016.20.59.png)

### **How Gift Folder Works**

Content inside a Gift Folder is always free and is used by AI as a gift for fans.

A gift can be sent in the following cases:

- if AI was chatting with a fan and the conversation ended with a goodbye
- if a fan sent Tips to thank the model for her content

Along with the content, AI also sends a message that accompanies the gift and makes the interaction feel more natural.
