## Макет

Панель розділена на три зони:

- **Бокова панель (ліворуч)** — навігація, перемикач моделей та елементи керування акаунтом.
- **Верхня панель** — кнопка згортання бокової панелі, пошук, перемикач теми та сповіщення.
- **Основна область** — вміст поточної сторінки.

## Навігація бокової панелі

Бокова панель має два режими залежно від того, де ви знаходитесь у застосунку.

### Режим агенції (за замовчуванням)

На верхньорівневій панелі бокова панель показує розділи всієї агенції:

| Розділ | Сторінки |
|--------|----------|
| **Огляд** | Панель (KPI та графіки по всіх моделях) |
| **Операції** | Моделі, Чати, Контент |
| **Аналітика** | Аналітика, AI-завдання |
| **Акаунт** | Інтеграції |

### Режим моделі

Коли ви відкриваєте конкретну модель, бокова панель перемикається на вкладки, специфічні для цієї моделі:

| Вкладка | Що показує |
|---------|------------|
| Огляд | Елементи керування та швидка статистика для цієї моделі |
| Чати | Усі розмови з фанатами |
| Профіль | Налаштування AI-персони та ідентичності |
| Контент | Бібліотека контенту на основі папок |
| Вітальний потік | Вітальні серії для нових підписників |
| Списки | Маршрутизація списків фанатів OnlyFans |
| Налаштування | Автоматизація, обмеження та небезпечна зона |

### Перемикач моделей

У верхній частині бокової панелі відображається назва поточної моделі (або «Всі моделі»). Натисніть, щоб:

- Знайти і перейти до іншої моделі.
- Повернутися до вигляду агенції (Всі моделі).

### Посилання в нижній частині панелі

Внизу бокової панелі:

- **Що нового** — читайте останні нотатки до випуску.
- **Додати модель** — запустити майстер підключення OnlyFans.
- **Вийти** — вийти з акаунту.

## Верхня панель

| Елемент | Призначення |
|---------|-------------|
| **Кнопка бокової панелі** | Згорнути або розгорнути бокову панель. |
| **Рядок пошуку** | Шукати фанатів, розмови або контент. |
| **Перемикач теми** | Перемкнутися між світлою та темною темою. |
| **Сповіщення** | Дзвіночок сповіщень. |
| **Аватар користувача** | Відображає ваші ініціали. |

## Головна сторінка панелі

Головна сторінка надає операційний огляд по всіх моделях:

- **Фільтр моделей** — оберіть одну або кілька моделей для звуження даних.
- **Вибір діапазону дат** — оберіть часовий проміжок для всіх показників.
- **Картки KPI** — загальна кількість моделей, фанатів, дохід, повідомлення, PPV-продажі та активні розмови.
- **Графіки часових рядів** — дохід та активність у динаміці.
- **Таблиця моделей** — деталізація по кожній моделі з накладеними даними про використання AI.

:::tip
Використовуйте фільтр моделей на головній панелі, щоб порівняти ефективність вибраних моделей пліч-о-пліч.
:::
