## Доступ до чатів

Є два способи відкрити папку вхідних чатів:

1. **З бокової панелі агенції** — натисніть **Чати** у розділі «Операції». Вас попросять спочатку вибрати модель, після чого буде відкрито папку вхідних для неї.
2. **З середини моделі** — натисніть вкладку **Чати** в боковій панелі моделі. Це одразу відкриє список розмов цієї моделі.

## Список розмов

Ліва панель показує всі розмови з фанатами для обраної моделі, відсортовані за часом останньої активності. Кожен запис містить:

- **Ім'я або ID фаната**
- **Попередній перегляд останнього повідомлення**
- **Відмітку часу**

Прокручуйте вниз, щоб завантажити більше розмов (список підтримує нескінченне прокручування).

Використовуйте поле **пошуку** у верхній частині списку для фільтрації розмов за іменем фаната.

## Гілка розмови

Натисніть на будь-яку розмову у списку, щоб відкрити повну гілку повідомлень праворуч.

У гілці відображаються:

- **Усі повідомлення** у хронологічному порядку, чітко позначені як повідомлення фаната або відповіді моделі.
- **PPV-повідомлення** вбудовані в гілку, щоб ви могли бачити, коли і який контент було надіслано.

### Надсилання повідомлення

Введіть текст у поле повідомлення внизу гілки та натисніть Enter (або кнопку «Надіслати»), щоб відправити повідомлення вручну.

:::note
Повідомлення, надіслані вручну через панель, відправляються з підключеного акаунту OnlyFans — так само, як і будь-яке інше повідомлення.
:::

### Переклад

Якщо фанат пише іншою мовою, використовуйте селектор **Мова перекладу** (відображається біля заголовка розмови), щоб перекласти повідомлення на зручну для вас мову. Це лише для вашого зручного читання — на те, що надсилається фанату, не впливає.

## Прямі посилання на конкретного фаната

Кілька частин панелі дають прямі посилання на розмову з фанатом із заздалегідь вибраним контекстом:

- Натискання на рядок фаната в таблиці PPV в розділі **Аналітика**.
- Натискання **Чат** у запиті контенту в розділі **AI-завдання**.
- Натискання посилання на фаната в **Журналах змін**.

Ці посилання відкривають сторінку Чатів для правильної моделі та прокручують безпосередньо до розмови з цим фанатом.

## Поради

- Якщо в розмові спостерігається несподівана поведінка, перевірте **Журнали змін**, щоб дізнатися, чи нещодавно змінювалися налаштування.
- Щоб призупинити відповіді для конкретної моделі, переключіть **Активна** в положення ВИМК у [Керуванні моделлю](/uk/models/controls/).
