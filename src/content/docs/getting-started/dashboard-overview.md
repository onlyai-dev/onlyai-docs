## Layout

The dashboard is split into three areas:

- **Sidebar (left)** — navigation, model switcher, and account controls.
- **Top bar** — sidebar toggle, search, theme switch, and notifications.
- **Main area** — the content for whichever page you're on.

## Sidebar navigation

The sidebar has two modes depending on where you are in the app.

### Agency mode (default)

When you're at the top-level dashboard, the sidebar shows agency-wide sections:

| Section | Pages |
|---------|-------|
| **Overview** | Dashboard (KPIs + charts for all models) |
| **Operations** | Models, Chats, Content |
| **Intelligence** | Analytics, AI Tasks |
| **Account** | Integrations |

### Model mode

When you open a specific model, the sidebar switches to model-specific tabs:

| Tab | What it shows |
|-----|--------------|
| Overview | Controls and quick stats for this model |
| Chats | All fan conversations |
| Profile | AI persona and identity settings |
| Content | Folder-based content library |
| Welcome Flow | New subscriber greeting sequences |
| Lists | OnlyFans fan list routing |
| Settings | Automation, limits, and danger zone |

### Model switcher

At the top of the sidebar you'll see the current model name (or "All Models"). Click it to:

- Search and switch to a different model.
- Return to the agency view (All Models).

### Footer links

At the bottom of the sidebar:

- **What's New** — read the latest release notes.
- **Add Model** — start the OnlyFans connection wizard.
- **Sign Out** — log out of your account.

## Top bar

| Element | Purpose |
|---------|---------|
| **Sidebar toggle** | Collapse or expand the sidebar. |
| **Search bar** | Search fans, conversations, or content. |
| **Theme toggle** | Switch between light and dark mode. |
| **Notifications** | Notification bell. |
| **User avatar** | Displays your initials. |

## Main dashboard page

The main dashboard gives you a cross-model operational overview:

- **Model filter** — select one or more models to scope the data.
- **Date range picker** — choose the time window for all metrics.
- **KPI cards** — total models, fans, revenue, messages, PPV sales, and active conversations.
- **Timeseries charts** — revenue and activity over time.
- **Models table** — per-model breakdown with AI usage overlaid.

:::tip
Use the model filter on the main dashboard to compare performance across selected models side-by-side.
:::
