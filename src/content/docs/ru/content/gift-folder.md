## **Gift Folder**

Gift Folder используется для отправки бесплатного контента фанатам в определённых ситуациях.

Для создания Gift Folder перейдите в раздел **Content → Gifts** и нажмите **Add Folder**.

![](/images/docs/gift-folder/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-06-04%20%D0%BE%2016.21.23.png)

При создании папки необходимо:

- выбрать медиафайлы через **Select Media**
- указать название папки
- заполнить описание и другие параметры

Для быстрого заполнения можно использовать функцию **AI Fill**, которая автоматически сгенерирует описание и заполнит необходимые поля на основе выбранного контента.

![](/images/docs/gift-folder/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-06-04%20%D0%BE%2016.20.59.png)

### **Как работает Gift Folder**

Контент в Gift Folder всегда является бесплатным и используется AI как подарок для фанатов.

Подарок может быть отправлен в следующих случаях:

- если AI вёл диалог с фанатом и разговор завершился прощанием
- если фанат отправил модели чаевые (Tips), чтобы поблагодарить его за поддержку

Вместе с контентом AI также отправляет сообщение, которое сопровождает подарок и делает взаимодействие более естественным.
