## Что такое PPV-папки?

PPV-папки (pay-per-view) — это коллекции контента, которые AI отправляет фанатам в виде платных предложений. Когда AI определяет, что фанат готов к покупке, он выбирает папку и доставляет контент по настроенной цене.

## Создание PPV-папки

1. Перейдите на вкладку **Контент** для нужной модели.
2. В группе **PPV** нажмите **Создать папку**.
3. В открывшемся окне заполните поля:
   - **Имя папки** — описательное внутреннее название (фанаты его не видят).
   - **Список хранилища** — список хранилища OnlyFans с медиафайлами для этой папки.
   - **Цена** — сколько фанат платит за этот контент.
4. Нажмите **Сохранить**.

Папка теперь доступна для использования AI.

## Ценовая лесенка

В группе PPV можно настроить **ценовую лесенку** — последовательность папок с разными ценами. AI использует эту лесенку для постепенного повышения предложений: начинает с более дешёвого контента и движется вверх.

Чтобы настроить лесенку, создайте несколько PPV-папок и расположите их в порядке возрастания цены.

:::tip
Хорошо структурированная лесенка обычно начинается с недорогой тизерной папки ($5–$10) и доходит до премиального контента ($30+). Это даёт AI естественные точки для продвижения.
:::

## Подпапки

Каждая PPV-папка может содержать **подпапки** для дополнительной организации вариантов контента (например, по теме, формату или интенсивности). AI выбирает из доступных подпапок на основе контекста.

Нажмите на папку, чтобы просмотреть её подпапки и добавить новые кнопкой **+**.

## Редактирование и удаление папки

Наведите курсор на любую папку и нажмите меню **⋯**, чтобы:
- **Редактировать** — изменить имя, связанный список хранилища или цену.
- **Удалить** — убрать папку из библиотеки OnlyAI. Это не удаляет контент из хранилища OnlyFans.

:::caution
Удаление PPV-папки означает, что AI больше не будет использовать этот контент. Если фанат недавно получил сообщение об этом контенте, последующие взаимодействия могут ссылаться на несуществующую папку.
:::

## Как AI выбирает PPV-контент

Когда для модели включены **Продажи**, AI учитывает структуру PPV-папок при выборе предложения для фаната. Он использует ценовую лесенку и контекст фаната (например, предыдущие покупки), чтобы подобрать наиболее подходящую папку.

→ Для включения продаж см. раздел [Управление моделью](/ru/models/controls/).
