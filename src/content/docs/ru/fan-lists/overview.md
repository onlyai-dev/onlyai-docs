## Что такое списки фанатов?

OnlyFans позволяет авторам организовывать подписчиков в **пользовательские списки** (например, «VIP-фанаты», «бесплатные фанаты», «крупные спонсоры»). Функция **Списки** в OnlyAI привязывает эти списки OnlyFans к **маршрутным дорожкам** — правилам, указывающим AI, как по-разному обращаться с фанатами из каждого списка.

Откройте вкладку «Списки» в боковой панели модели.

## Зачем нужна маршрутизация?

Разные фанаты заслуживают разного отношения:

- **VIP-фанаты** — крупные спонсоры, которые должны получать премиальный контент и более персонализированное внимание.
- **Бесплатные подписчики** — фанаты на бесплатном пробном периоде, которых нужно мягко направлять к платному контенту.
- **Ушедшие фанаты** — бывшие подписчики, которых, возможно, стоит попытаться вернуть.

Без маршрутизации AI относится ко всем фанатам одинаково. С маршрутизацией он адаптирует поведение в зависимости от того, в каком списке находится фанат.

## Настройка маршрутной дорожки

:::note
Перед настройкой маршрутных дорожек убедитесь, что в аккаунте OnlyFans созданы пользовательские списки. Они управляются непосредственно в OnlyFans, а не в OnlyAI.
:::

1. Перейдите на вкладку **Списки** для нужной модели.
2. Каждая **дорожка** представляет правило маршрутизации. Нажмите **Настроить** рядом с нужной дорожкой.
3. Привяжите дорожку к одному или нескольким пользовательским спискам OnlyFans с помощью выборщика списков.
4. Сохраните конфигурацию.

## Количество фанатов в дорожках

На странице «Списки» отображается текущее количество фанатов для каждого подключённого списка. Данные загружаются в реальном времени из OnlyFans и дают быстрое представление о том, сколько фанатов проходит через каждую дорожку.

## Подключение и отключение списков

Чтобы изменить, какой список OnlyFans привязан к дорожке:

- Нажмите **Подключить** или **Редактировать** рядом с дорожкой.
- Выберите новый список из выпадающего меню.
- Сохраните.

Чтобы убрать список из дорожки, нажмите **Отключить** рядом с привязанным списком.

:::tip
Если фанат входит в несколько списков OnlyFans, он будет направлен по первой подходящей дорожке в порядке приоритета. Пересмотрите приоритеты дорожек, если фанаты маршрутизируются неожиданным образом.
:::
