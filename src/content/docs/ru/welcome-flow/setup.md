## Что такое приветственный поток?

Приветственный поток — это серия автоматических сообщений, отправляемых фанату при первой подписке. Это первое впечатление от AI — и если всё настроено правильно, это существенно повышает ранние вовлечённость и конверсию в PPV.

Приветственный поток настраивается для каждой модели на вкладке **Приветственный поток**.

## Включение приветственного потока

Прежде чем настраивать сообщения, убедитесь, что приветственный поток включён:

- На странице **Обзора** или **Настроек** модели переключите **Приветственный поток** в положение ВКЛ.

:::note
Даже если приветственный поток полностью настроен, он не будет запускаться, пока этот переключатель выключен.
:::

## Настройка приветственного сообщения

Приветственное сообщение имеет два варианта в зависимости от того, известно ли OnlyAI имя фаната:

| Вариант | Когда используется |
|---------|-------------------|
| **Имя известно** | Имя фаната доступно из OnlyFans. Сообщение можно персонализировать. |
| **Без имени** | Имя фаната недоступно. Используется тёплое обращение без имени. |

1. Перейдите на вкладку **Приветственный поток** для нужной модели.
2. Найдите раздел **Приветственное сообщение**.
3. Нажмите **Редактировать** рядом с вариантом **Имя известно** и напишите персонализированную версию. Используйте `{name}` как заглушку для имени фаната.
4. Нажмите **Редактировать** рядом с вариантом **Без имени** и напишите запасной вариант.
5. Нажмите **Сохранить** для каждого варианта.

:::tip
Пишите приветственное сообщение в голосе модели, с учётом её персоны. Делайте его коротким, тёплым, с мягким призывом к действию (например, «загляни в мой новый контент» или «расскажи мне о себе»).
:::

## Варианты сообщений

Для каждого типа приветственного сообщения можно добавить несколько вариантов. AI будет чередовать их, чтобы избежать повторений для фанатов, которые могут видеть похожие сообщения на разных аккаунтах.

Нажмите **Добавить вариант**, чтобы создать дополнительную версию сообщения.

## Сохранение

Каждый раздел сообщений сохраняется независимо. Нажимайте **Сохранить** после редактирования каждого варианта, чтобы сохранить изменения.
