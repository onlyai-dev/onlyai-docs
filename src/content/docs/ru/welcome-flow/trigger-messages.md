Trigger messages — это функция автоматических сообщений, которые бот отправляет при лайке от фаната или гогда он онлайн, чтобы мягко вернуть пользователя в диалог и повысить вовлечённость.

Функция работает с учётом временных ограничений, чтобы избежать избыточной активности со стороны бота.

<aside> 💡

Перед отправкой бот проверяет

- не было ли сообщений от бота за последние&nbsp;**4 часа**
- не было ли отправленных Trigger-сообщений за последние&nbsp;**8 часов**

Если оба условия соблюдены → бот отправляет trigger-сообщение.

</aside>

### **Как выбирается сообщение**

- сообщение выбирается&nbsp;**случайным образом из списка добавленных**
- приоритет отдаётся сообщениям, которые ранее не отправлялись
- после использования всех сообщений начинается повторный цикл

## **Шаги настройки**

### **Шаг 1.** Откройте модель → вкладка&nbsp;Trigger **messages**

## **Шаг 2. Добавить Trigger-сообщения**

Есть два способа добавления

### **Вариант 1. Import**

Нажмите&nbsp;**Import**, чтобы выбрать сообщения из предложенных вариантов.

- можно использовать готовые сообщения
- можно взять их за основу и отредактировать под модель

![](/images/docs/trigger-messages/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-05-28%20%D0%BE%2009.34.01.png)

При нажатии на Import выбранный Trigger автоматически добавляется в список

### **Вариант 2. Add**

Нажмите&nbsp;**Add**, чтобы создать собственное сообщение.

Доступные варианты настройки:

- можно добавить&nbsp;**только текстовое сообщение**
- можно добавить&nbsp;**медиафайл**
- при добавлении медиа можно:
  - установить&nbsp;**цену**&nbsp;(если контент платный)
  - добавить&nbsp;**preview**&nbsp;для предпросмотра
- текст и медиа можно использовать как вместе, так и по отдельности

![](/images/docs/trigger-messages/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-05-28%20%D0%BE%2009.35.00.png)

После завершения настройки нажмите&nbsp;**Create Trigger,** созданное сообщение автоматически появится в общем списке

### **Шаг 3. Активировать функцию**

После добавления достаточного количества Trigger-сообщений необходимо активировать функцию.

![](/images/docs/trigger-messages/%D0%97%D0%BD%D1%96%D0%BC%D0%BE%D0%BA%20%D0%B5%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-05-28%20%D0%BE%2009.36.00.png)

Рекомендуется перед активацией убедиться, что список сообщений заполнен:

- 4-5 текстовых сообщений
- 1–2 сообщения с медиа (или больше, при наличии контента)

Это позволит боту использовать разные варианты сообщений
