## Каталог моделей

Страница **Модели** отображает все аккаунты OnlyFans, подключённые к вашему агентству. С неё начинается работа с любой отдельной моделью.

### Просмотр моделей

Список можно просматривать в двух режимах:

- **Сетка** — карточки с аватаром модели, именем, статусом и быстрыми переключателями.
- **Список** — компактная таблица для быстрого просмотра большого количества моделей.

Предпочтительный режим отображения сохраняется автоматически.

### Фильтрация и поиск

- **Поиск** — введите текст в строку поиска, чтобы отфильтровать модели по имени.
- **Вкладки статуса** — фильтрация по статусу: Активные, Неактивные или Все.
- **Фильтр агентства** — если у вас есть доступ к нескольким агентствам, переключайтесь между ними.

### Статус модели с первого взгляда

На каждой карточке или строке отображается, активна ли модель в данный момент (работает ли AI-бот). Переключить состояние активности можно прямо из списка моделей, не открывая её.

## Открытие модели

Нажмите на любую карточку или строку, чтобы открыть рабочее пространство этой модели. Боковая панель переключится на навигацию для данной модели и откроет страницу **Обзора**.

## Добавление модели

Нажмите **Add Model** в верхней части  панели  на странице «Модели». Откроется мастер подключения OnlyFans.

→ Полное руководство см. в разделе [Подключение аккаунта OnlyFans](/ru/models/connect-onlyfans/).

:::note
Для добавления новых моделей в агентство необходима соответствующая роль (администратор, суперадминистратор или руководитель команды).
:::
