## Доступ к аналитике

Перейдите в **Аналитика** в боковой панели агентства (раздел **Аналитика**). Откроется полная панель доходов и статистики.

:::note
Страница **Аналитика** показывает актуальные данные из подключённых моделей. Убедитесь, что вы просматриваете корректный раздел, а не демо-версию.
:::

## Фильтры

В верхней части страницы можно уточнить данные:

- **Модель** — выберите конкретную модель или просматривайте все модели суммарно.
- **Период** — выберите временной диапазон для анализа.
- **Агентство** *(только для администраторов)* — фильтрация по нескольким агентствам.

## Метрики доходов

В верхнем разделе отображается доход с разбивкой по типам:

| Метрика | Что измеряет |
|---------|-------------|
| **PPV-доход** | Суммарный заработок от PPV-сообщений. |
| **Доход от чаевых** | Суммарный заработок от прямых чаевых. |
| **Общий доход** | PPV + чаевые суммарно. |

Данные отображаются в виде итоговых значений и **столбчатой диаграммы с накоплением** по времени — видны ежедневные тенденции.

## Метрики активности и эффективности

Ниже доходов находятся операционные метрики:

- **Отправлено сообщений** — общее количество исходящих сообщений от AI.
- **Начато переписок** — новые треды, открытые за период.
- **Конверсия PPV** — доля фанатов, совершивших покупку после получения PPV-предложения.
- **Эффективность AI** — соотношение сгенерированного дохода к отправленным сообщениям.

## Воронка фанатов

Визуализация воронки показывает отсев на каждом этапе пути фаната:

1. Новые подписчики
2. Фанаты, получившие сообщение
3. Фанаты, ответившие
4. Фанаты, получившие PPV-предложение
5. Фанаты, совершившие покупку

Используйте воронку для выявления этапов с наибольшим отсевом и оптимизации соответствующих шагов.

## Таблица PPV-сообщений

В нижней части страницы находится детальная таблица всех PPV-сообщений за период. Колонки включают:

- Имя / ID фаната
- Папка контента
- Цена
- Статус (отправлено / куплено / истекло)
- Временна́я метка

Нажмите на любую строку, чтобы перейти прямо к переписке с этим фанатом в разделе **Чаты**.

### Обновление таблицы

Нажмите кнопку **Обновить**, чтобы загрузить последние данные. Таблица может немного отставать от реального времени.

### Пагинация

Используйте элементы пагинации в нижней части таблицы для навигации по большим наборам данных.

:::tip
Отсортируйте таблицу PPV по убыванию цены, чтобы быстро найти наиболее ценные транзакции.
:::
