OnlyAI — это платформа управления с поддержкой искусственного интеллекта для авторов OnlyFans и агентств. Она напрямую подключается к вашему аккаунту OnlyFans и автоматизирует наиболее трудоёмкие аспекты ведения бизнеса автора: общение с фанатами, доставку PPV-контента, приветствие новых подписчиков и отслеживание дохода.

## Для кого предназначен OnlyAI?

OnlyAI создан для двух типов пользователей:

- **Авторы и операторы**, которые управляют одним или несколькими аккаунтами OnlyFans и хотят автоматизировать взаимодействие с фанатами и доставку контента.
- **Команды агентств**, где руководитель или менеджер контролирует нескольких моделей, распределяет работу между чаттерами и следит за показателями по всему портфелю.

## Что автоматизирует OnlyAI?

| Функция | Описание |
|---------|----------|
| **AI-чат** | Автоматически отвечает фанатам, используя обученную персону на основе профиля модели. |
| **Доставка PPV** | Определяет, когда фанат готов купить, и отправляет нужный контент из вашей библиотеки. |
| **Приветственный поток** | Встречает новых подписчиков персонализированной последовательностью сообщений с опциональными продолжениями. |
| **Прогрев-рассылка** | Отправляет серию сообщений для вовлечения фанатов, которые давно не проявляли активности. |

## Структура панели управления

Панель управления OnlyAI имеет два основных контекста:

- **Агентский вид** — верхнеуровневая панель, где отображаются все модели, общий доход, использование AI и настройки команды.
- **Вид модели** — рабочее пространство для отдельной модели с вкладками: Чаты, Контент, Профиль, Приветственный поток, Списки и Настройки.

Переключение между моделями осуществляется через переключатель моделей в боковой панели.
