## Структура интерфейса

Панель управления разделена на три области:

- **Боковая панель (слева)** — навигация, переключатель моделей и управление аккаунтом.
- **Верхняя панель** — переключатель боковой панели, поиск, смена темы и уведомления.
- **Основная область** — содержимое текущей страницы.

## Навигация в боковой панели

Боковая панель работает в двух режимах в зависимости от того, в каком разделе приложения вы находитесь.

### Переключатель моделей

В верхней части боковой панели отображается имя текущей модели (или «Все модели»). Нажмите на него, чтобы:

- Найти и переключиться на другую модель.
- Вернуться к агентскому виду (Все модели).

### Нижние ссылки

В нижней части боковой панели:

- **Что нового** — ознакомьтесь с последними обновлениями.
- **Добавить модель** — запустить мастер подключения OnlyFans.
- **Выйти** — выйти из аккаунта.

## Главная страница панели управления

Главная страница даёт операционный обзор по всем моделям:

- **Фильтр по моделям** — выберите одну или несколько моделей для фильтрации данных.
- **Выбор периода** — выберите временной диапазон для всех показателей.
- **KPI-карточки** — общее количество моделей, фанатов, доход, сообщения, PPV-продажи и активные переписки.
- **Временные графики** — доход и активность с течением времени.
- **Таблица моделей** — разбивка по моделям с наложенными данными об использовании AI.

:::tip
Используйте фильтр по моделям на главной странице, чтобы сравнивать показатели выбранных моделей бок о бок.
:::
