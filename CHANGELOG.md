# Changelog

## Апрель 2026

### Добавлено

**Фильтр по моделям в разделе Content Requests**

Добавлен фильтр по моделям в формате выпадающего списка.

- можно выбрать только одну модель

- значение по умолчанию — «All models» (показывает все запросы)

- список моделей отсортирован по алфавиту

Результат:

- быстрее находить запросы по конкретной модели

- удобнее отслеживать повторяющиеся запросы

- упростился анализ контент-запросов

---

**Сортировка по времени в Content Requests**

Добавлена возможность сортировки по колонке Time.

- можно переключаться между новыми и старыми запросами

- по умолчанию отображаются самые новые запросы

Результат:

- проще работать с историей запросов

- быстрее находить старые записи без ручного поиска

---

**График Absolute Metrics в разделе Stats**

Добавлен график с основными метриками.

Включает:

- Total, $

- PPV Sales, $

- Tips, $

Функциональность:

- выбор любого периода (date range)

- выбор одной модели или всех моделей агентства

- включение/выключение метрик

- возможность отображать несколько метрик одновременно

- тултип с точным значением при наведении

Результат:

- основные метрики отображаются в одном месте

- улучшено понимание структуры дохода

- упрощён анализ динамики

---

**Графики конверсий (PPV Funnel и Fans Funnel)**

Добавлены графики для анализа конверсий.

PPV Funnel:

- Engaged → PPV Sent CR

- PPV Sent → PPV Purchased CR

- Engaged → PPV Purchased CR

Fans Funnel:

- New → Engaged CR

- Engaged → Paying CR

- New → Paying CR

Дополнительно:

- автоматический расчет всех конверсий

- выбор периода

- выбор модели или всех моделей

- включение/выключение метрик

- возможность отображать несколько метрик одновременно

- тултип с точными значениями

Результат:

- появилась наглядная воронка конверсий

- проще оценивать эффективность работы бота

- легче находить узкие места

---

**Переименование раздела Models list → Fans List**

Раздел переименован для соответствия фактическому содержимому.

Результат:

- уменьшена путаница в интерфейсе

- структура стала более логичной

- быстрее ориентироваться в системе

---

**Переименование кнопки Generate with AI → Fill with AI**

Изменено название кнопки во всех местах интерфейса.

Причина:

- старое название вводило в заблуждение (создание нового контента)

- фактически используется для заполнения существующего

Результат:

- точнее отражает действие

- упрощает понимание функционала

- снижает количество ошибок при использовании
