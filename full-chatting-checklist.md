## **Этап 4. Создать 3 PPV папки (10 минут)**

PPV папки — это база платного контента, с которой бот работает в диалоге.

**Что нужно для выполнения шага:**

Контент от модели разнообразной тематики (например: селфи, флирт-контент, tease, более откровенные фото/видео).

**Шаги:**

1. Нажмите Models
2. Выберите модель, для которой хотите добавить контент
3. Нажмите Folders
4. Найдите блок PPV Folders и Нажмите Add PPV folder
5. В блоке Media content нажмите кнопку Select Media
6. Выберите папку с контентом из OnlyFans (vault), которую хотите использовать для PPV
7. Заполните поля PPV папки:
    - Name
    - Description (1–2 строки)
    - Price
    - Tags (3–5 тегов)
8. При необходимости можно:
    - включить тоггл On request only — папка не отправляется автоматически, а используется только в ответ на запрос фана (например, если он просит конкретный тип контента)
    - добавить Preview — чтобы фан видел превью перед покупкой (повышает вероятность открытия и покупки)
    - отметить Tagged users — если в контенте есть другие люди

> 💡 Облегчить заполнение можно, нажав на кнопку Generate with AI. Затем при необходимости отредактировать.

9. Повторите действия для создания 3 PPV папок

**Что получилось:**

Создана база платного контента, с которой бот может работать.

## **Этап 5. (ОПЦИОНАЛЬНО) Настроить 1 Ladder (10 минут)**

Если настроить хотя бы 1 Ladder, боту будет проще вести диалог и он сможет торговаться с фанатами о цене.

**Что нужно для выполнения шага:**

1. 2-3 дополнительные PPV-папки с контентом (помимо основной)
2. Контент разной ценности (от более простого к более откровенному)

**Шаги:**

1. Найдите блок PPV Folders и Нажмите Add PPV folder
2. В блоке Media content нажмите кнопку Select Media
3. Выберите папку с контентом из OnlyFans (vault), которую хотите использовать для ladder
4. Включите Ladder mode (subfolders)
5. Нажмите кнопку Create PPV folder
6. В списке папок появится значок Ladder у созданной папки

<img width="535" height="368" alt="Screenshot%202026-04-15%20at%2015 07 39" src="https://github.com/user-attachments/assets/2aba3b58-643f-4b62-b324-ca0040fffde7" />


7. Выберите Manage Subfolders
8. Нажмите на кнопку Add Subfolder

> 💡 Создайте минимум 2 под-папки (это необходимо для активации Ladder) по нажатию на кнопку Add Subfolder.

9. Настройте под-папку:
- Нажмите Select media
- Выберите контент
- Добавьте превью (Select previews)
- Укажите Description
- Укажите Price

> 💡 При необходимости включите Enable discount offer — если вы хотите, чтобы бот мог торговаться с фаном.

10. Настройте Warmup folder
- Нажмите Manage warmup folder
- Нажмите Add Warmup
- Выберите медиа

> 💡 Уже использованные фото будут отображаться темнее, чтобы избежать повторов.

## Этап 6. (ОПЦИОНАЛЬНО) Настроить Welcome folder (2 минуты)

Welcome folder отправляется новым фанатам сразу после подписки на модель. Бот отправляет все медиафайлы из папки сразу (всю папку).

**Что нужно для выполнения шага:**

Фото и/или видео для Welcome.

**Шаги:**

1. Нажмите Models  
2. Выберите модель, для которой хотите добавить контент  
3. Нажмите Folders  
4. Найдите блок Welcome folder и Нажмите Set up  
5. В открывшемся окне нажмите Select media  
6. Выберите папку для Welcome из списка папок  
7. Установите цену  
   - Оставьте поле Price пустым → папка будет бесплатной  
   - Укажите цену → папка станет платной  
8. Добавьте бесплатные превью  
   - Доступно только для платных папок  
   - Нажмите Select previews  
   - Выберите медиа для предпросмотра  

> 💡 При добавлении бесплатного превью фан видит часть контента бесплатно. Полный доступ остаётся платным.

9. Сохраните изменения, нажав Save  

❗**Важно**: Без сохранения настройки не применятся и бот не сможет использовать папку.

## Этап 10. (ОПЦИОНАЛЬНО) Настроить VIP Club (2 минуты)

VIP Club — это платный доступ для фанов (разовая оплата через tip), который даёт им доступ к закрытой папке с контентом и приоритет в общении.

**Что нужно для выполнения шага:**

Папка с VIP-контентом

**Шаги:**

1. Нажмите Models  
2. Выберите модель, для которой хотите настроить VIP Club  
3. Перейдите в раздел Settings  
4. Найдите блок VIP Club  
5. Включите Enable VIP Club  
6. Выберите папку в поле Folder  

> 💡 Это папка, к которой получат доступ только VIP-фаны

7. Укажите стоимость в поле Price ($)  

> 💡 Это сумма, которую фан должен отправить через tip

8. Заполните поле Description. Укажите, что получает фан (контент, доступ, приоритет и т.д.)  
9. Сохраните изменения, нажав Save VIP Club Settings  


**Что получилось:**

VIP Club настроен и готов к работе.

## Этап 11. (ОПЦИОНАЛЬНО) Настроить Bump messages (5 минут)

Bump messages — это функция, которая отправляет автоматические сообщения при лайке от фаната. Помогает вернуть пользователя в диалог и повысить вовлечённость.

**Что нужно для настройки:**

- подготовить тексты для сообщений или использовать готовые варианты через Import  
- подготовить новые папки с контентом (при необходимости)

**Шаги:**

**Шаги настройки:**

1. Нажмите Models  
2. Выберите модель, для которой хотите настроить Bump messages  
3. Перейдите во вкладку Bump message  
4. Добавьте сообщения одним из способов:  
   - через Import — выбрать из предложенных вариантов  
   - через Add — создать своё сообщение  
5. При добавлении через Add:  
   - напишите текст сообщения  
   - при необходимости добавьте медиафайл  
   - установите цену (опционально)  
   - добавьте preview  
   - нажмите Create Bump  
6. Убедитесь, что добавлено несколько сообщений (текст + медиа)  
7. Включите функцию
